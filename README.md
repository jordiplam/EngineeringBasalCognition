# Engineering Basal Cognition

**"Engineering Basal Cognition: Minimal Genetic Circuits for Habituation, Sensitization, and Massed–Spaced Learning"**  
*Jordi Pla-Mauri and Ricard Solé*  
*ACS Synthetic Biology* (2025)  
DOI: [10.1021/acssynbio.5c00766](https://doi.org/10.1021/acssynbio.5c00766)

A preprint version of the manuscript is also freely available in bioRxiv: [10.1101/2025.09.09.674925](https://doi.org/10.1101/2025.09.09.674925)
